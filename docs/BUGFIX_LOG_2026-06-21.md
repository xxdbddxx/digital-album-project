# 2026-06-21 语音与媒体链路更新日志

## 语音输入

- 完成 INMP441 麦克风采集、WakeNet 唤醒词、VAD、WebSocket 音频上传链路。
- 唤醒词统一为“你好小智”，并保留串口按键 `7` 强制唤醒调试入口。
- faster-whisper 升级为 `large-v3-turbo`，使用 CUDA `float16` 和 beam size 5。
- 修复录音固定等待较长时间的问题，用户停止说话约 1.2 秒后自动结束。
- 端点检测改为稳定语音能量参考，避免瞬时峰值或尾部环境噪声持续重置静音计数。
- 修复 WebSocket 异常断开后录音批处理残留污染下一轮会话的问题。

## 连续对话与唤醒

- 连续对话触发阈值参考上一轮用户真实语音平均 RMS。
- 修复环境底噪超过固定阈值后误进入连续录音的问题。
- 修复连续对话等待状态阻塞照片命令轮询约 30 秒的问题。
- 音乐播放期间保持 WakeNet 工作，可通过“你好小智”打断播放。
- 修复歌曲人声段无法唤醒的问题，调整 AEC 输入增益和播放参考时序。

## LLM、TTS 与界面

- LLM 接入 DeepSeek V4 Flash，支持整句语义分析照片、方向、音乐等组合指令。
- TTS 接入阿里云 CosyVoice，并保留 Edge TTS 和 Piper TTS 兜底。
- 回复文字在生成对应 TTS 前发送到 ESP32，使对话框和语音同步开始。
- 修复完整 CJSON 到达后重复启动打字机效果的问题。
- 修复 WebSocket 异常退出后语音助手遮罩不消失的问题。
- 修复对话框中文字乱码、协议 JSON 字段误显示和字库缺字日志问题。

## 音乐播放

- 支持本地音乐文件转换为适合 ESP32 的 16 kHz PCM 流。
- 默认播放音量调整为 60%，音乐默认不循环。
- 支持按文件名、内嵌标题和歌手元数据查找歌曲。
- 修复《晴天》等文件名与内嵌标题不同导致检索错误的问题。
- 明确指定歌曲不存在时不再随机播放其他歌曲。
- 支持“随机播放一首音乐”等明确的随机播放请求。
- 修复音乐长连接与照片心跳、命令轮询争抢内部网络内存导致的 `HTTP_CLIENT: Allocation failed`。

## 设备控制

- 照片切换和横竖屏方向可在同一条自然语言指令中同时执行。
